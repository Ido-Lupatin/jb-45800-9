## 02 - HTML - Gardening Website

יצירת אתר חדש עבור טיפוח גינות ומידע על גינון.

### 📝 דרישות פונקציונליות

## 03 - HTML - My Website
### 🎯 מטרה
בחירת נושא חופשי (אוכל / ביגוד / ספורט / משחקים / סרטים וכד') ובניית דף HTML מקיף.

### 📝 דרישות פונקציונליות
יש לשלב בדף לפחות פעם אחת את **כל** התגיות הבאות עם המאפיינים המתאימים:
- [ ] Structural: `<html>`, `<head>`, `<body>`, `<title>`
- [ ] Typography: `<h1>` עד `<h6>`, `<p>`, `<span>`, `<br>`, `<hr>`
- [ ] Media & Links: `<a href>`, `<img src>`, `<iframe>`
- [ ] Lists: `<ul>`, `<ol>`, `<li>`
- [ ] Containers: `<fieldset>`, `<legend>`
- `<table>` מלאה הכוללת:
  - [ ] `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`

## 05 - HTML - Software Programming Website
### 🎯 מטרה
יצירת אתר בנושא שפות תכנות המורכב מ-4 דפים המקושרים ביניהם (תפריט ניווט אחיד).

### 📝 מבנה קבוע לכל הדפים
- כותרת זהה בראש כל דף
- קו מפריד מתחת לכותרת
- תפריט ניווט (נתיבים יחסיים) המכיל 4 לינקים קבועים:
  1. דף הבית (`home.html`)
  2. הסבר על HTML (`hypertext-markup-language.html`)
  3. הסבר על CSS (`cascading-style-sheet.html`)
  4. הסבר על JavaScript (`javascript.html`)

### 📂 פירוט דפים ותוכן ייעודי:
- [ ] **home.html:** פסקת הסבר כללי על שפות תכנות + תמונה רלוונטית.
- [ ] **hypertext-markup-language.html:** פסקת הסבר על HTML + סרטון יוטיוב בנושא.
- [ ] **cascading-style-sheet.html:** פסקת הסבר על CSS + סרטון יוטיוב בנושא.
- [ ] **javascript.html:** פסקת הסבר על JS + סרטון יוטיוב בנושא.
